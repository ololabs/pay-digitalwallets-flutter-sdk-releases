ErrorCodes

ErrorCodes class

A list of all possible error codes that can be used in conjunction with PlatformException

Olo Digital Wallets SDK methods can throw a PlatformException. If they do, they will contain a code property indicating the cause for the exception. The possible error codes are defined in this class.
