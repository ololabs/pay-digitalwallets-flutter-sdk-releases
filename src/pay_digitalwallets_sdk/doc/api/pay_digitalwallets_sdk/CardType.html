CardType

CardType enum

Enum representing credit card types

For more general information on card types supported by each platform see:

  • Apple Pay - PKPaymentNetwork
  • Google Pay - CardParameters.allowedCardNetworks

Important: When submitting card details to Olo's ordering API the card type value passed should be the oloDescription of this enum.

Supported card types by Olo:

  • Visa
  • Mastercard
  • American Express
  • Discover
  • JCB
  • Diners Club
  • Union Pay
