# pay_digitalwallets_sdk_platform_interface

The `pay_digitalwallets_sdk_platform_interface` package defines the common platform interface for Olo's Digital Wallets SDK federated plugin.

This interface allows platform-specific implementations of the [pay_digitalwallets_sdk](main_package_link) plugin.

This package is not intended to be used directly. Instead, use the [pay_digitalwallets_sdk](main_package_link) package, which includes this package as a dependency.

For setup instructions, usage examples, and API documentation, please refer to the [pay_digitalwallets_sdk][main_package_link]
and [pay_digitalwallets_sdk_platform_interface](interface_package_link) documentation.

[main_package_link]: https://pub.dev/packages/pay_digitalwallets_sdk
[interface_package_link]: https://pub.dev/packages/pay_digitalwallets_sdk_platform_interface
