# pay_digitalwallets_sdk_ios

The iOS implementation of `pay_digitalwallets_sdk`.

## Usage

This package is [endorsed][endorsed_link], which means you can simply use [pay_digitalwallets_sdk](main_package_link)
normally. This package will be automatically included in your app when you do.

For setup instructions, usage examples, and API documentation, please refer to the [pay_digitalwallets_sdk][main_package_link]
and [pay_digitalwallets_sdk_platform_interface](interface_package_link) documentation.

[endorsed_link]: https://flutter.dev/docs/development/packages-and-plugins/developing-packages#endorsed-federated-plugin
[main_package_link]: https://pub.dev/packages/pay_digitalwallets_sdk
[interface_package_link]: https://pub.dev/packages/pay_digitalwallets_sdk_platform_interface
